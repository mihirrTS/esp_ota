Building Firmware (.bin files)
Arduino IDE:
  1. Open your ESP32 project in Arduino IDE
  2. Go to Sketch â†’ Export compiled Binary
  3. The .bin file will be created in your sketch folder
PlatformIO:
  1. Build your project: pio run
  2. Find the .bin file in .pio/build/[environment]/firmware.bin
ESP-IDF:
  1. Build your project: idf.py build
  2. Find the .bin file in build/[project_name].bin
